# Edu-Ability-26-CSI-RAIT-Cosmos

Edu-Ability is an innovative IoT and AI-driven ecosystem designed to bridge communication gaps for the deaf and hard-of-hearing community. 

## Core Features
- **Real-Time Translation**: Smart gloves equipped with flex sensors and IMUs translate sign language into text and speech.
- **AI-Powered Insight**: Uses a hybrid LSTM and XGBoost model for high-accuracy gesture recognition.
- **Accessibility Focus**: Designed with features specifically for dyslexic and neurodivergent users.
